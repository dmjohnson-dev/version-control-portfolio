# Version Control Portfolio Project

This repository contains a static website project used to demonstrate version control concepts, including repository structure, commits, and project organization.

## Project Overview

The project consists of a multi-page static website built with HTML, CSS, and JavaScript. It includes multiple content pages and supporting assets to demonstrate how version-controlled web projects are structured and maintained.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Git / GitHub

## Purpose

This repository serves as a **portfolio version** of a version control project originally completed as part of Western Governors University coursework.

The focus of this project is:
- Using Git to track changes
- Managing a multi-file project
- Maintaining a clean repository structure
- Publishing and syncing repositories to a remote host

## Notes on Commit History

Some early commits reflect the initial project setup. Subsequent commits represent my work preparing this repository for portfolio presentation and public hosting.

## How to View

Open `index.html` in a web browser to view the site locally.
